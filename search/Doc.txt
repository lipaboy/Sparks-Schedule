Документация

    Есть onetime - когда один старший, один дух
    Есть parttime - когда один старший, один дух и один дух на полусмене
    Есть pairday - когда один старший, два духа на полных сменах

    Сейчас реализовано:
        - parttime может быть только на pairday
        - pairday начинается с pairDayStart и заканчивается вск
            (т.е. для праздников в середине недели не получиться сделать отдельно)


    Подсчёт количества тачек:
        Нужно считать кроме количества тачек также количество смен.
        Количество смен считается в вещественном виде (так как бывают полусмены)
